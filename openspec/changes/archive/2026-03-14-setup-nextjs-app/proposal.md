## Why

This project needs a Next.js application foundation to serve as the starting point for a portfolio site. Establishing the app scaffold now provides a working baseline that future features can build on.

## What Changes

- Initialize a new Next.js application at the project root
- Configure TypeScript, Tailwind CSS, and standard Next.js conventions
- Add a single home page that renders "Hello world"

## Capabilities

### New Capabilities
- `hello-world-page`: A single root page (`/`) that renders "Hello world" text to the screen

### Modified Capabilities

## Impact

- Creates the Next.js app structure at the project root (package.json, app directory, config files)
- No existing code is affected — this is a greenfield setup
- Introduces dependencies: `next`, `react`, `react-dom`, TypeScript tooling
