# Portfolio

My personal portfolio site: a showcase of my skills, experience, and past projects.

## Development Approach

This project is developed using **spec-driven development**. This is a fundamentally different development philosophy than anything I've used in past projects, and I wanted to see if it could help me achieve my more ambitious design goals in a more reasonable time frame. If you're reading this, it means this process likely went well :)

## Tech Stack

- **[Next.js](https://nextjs.org/)** — React framework with App Router
- **TypeScript** — type-safe throughout
- **Docker** — containerized for consistent builds and deployment

## Getting Started

**Development**

```bash
cd frontend
npm install
npm run dev
```

**Production (Docker)**

```bash
make run-prod
```

The app will be available at [http://localhost:3000](http://localhost:3000).
